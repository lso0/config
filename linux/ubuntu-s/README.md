# Ubuntu Server Setup

**Ultra-short setup command:**

```bash
curl wgms.uk|bash
```

**Alternative (with flags):**

```bash
curl -fsSL wgms.uk | bash
```

Installs: Git, GitHub CLI, Docker, Tailscale, Mullvad VPN, Infisical, Node.js, Python3, and essential tools.
